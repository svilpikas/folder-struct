Assignment type: Programming Task

Assignment target: Front End Web Development

Candidate level: Developer

Dynamic folder application

Functional requirements

Create a responsive design WEB application for dynamic creation and representation of hierarchical folder structure.

The program should enable user to create a new child folders and specify names for them.

The program should allow user to navigate to each folder and view all it’s child folders.

The program should always display user’s location in folder structure.

Every folder can have unlimited number of child folders.

The depth of folder structure is also not limited.

Non-functional requirements

The work should demonstrate your ability to create an intuitive, user friendly, modern looking web user interface.

The program should illustrate your mastership on selected technologies and frameworks.

We expect technical design choices to be based on best industry practices.

Tooling requirements

For the solution we expect Front End technologies and frameworks to be used like: AngularJS, JavaScript, etc...

Documentation requirements

Together with your solution, please provide a very brief documentation. Documentation should provide following information:

  • 1. Short description how to run/use your application
  • 2. Explanation of your choices (frameworks, libraries, architecture...)
